# Lawn Mower Billionaire

A fast-paced arcade game where you cut grass, earn money, and avoid dangerous tractors on your way to becoming a billionaire.

## 🎮 About the Game

Move your lawn mower across the field to cut grass and earn euros automatically. The goal is simple: reach €1,000,000,000 without getting hit by tractors.

- Cut grass to earn money
- Avoid moving tractors
- Survive until you become a billionaire

## ▶️ Run Locally

1. Install dependencies:

```
npm install
```

2. Start the development server:

```
npm run dev
```

3. Open in browser:

```
http://localhost:8080
```
